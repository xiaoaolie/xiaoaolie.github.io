Idea连接mysql遇到的问题

如果Idea提示如下讯息
1.Server returns invalid timezone. Go to ‘Advanced’ tab and set ‘serverTimezone’ property manually.
那么小伙伴的安装mysql的时候肯定没有改变时区。
这句话的意思是:
服务器返回无效时区。转到“高级”选项卡并手动设置“serverTimezone”属性。
需要去设置mysql的时区.
所以打开CMD 输入mysql -hlocalhost -uroot -p
打开mysql，如果没有提示没有这个命令，说明小伙伴没有配置环境变量。
在mysql输入 ：
show variables like ‘%time_zone’; 记得加分号哦！

我的显示的是SYSTEM，意思就是没有设置时区。

设置时区命令:set time_zone = ‘+8:00’;

成功，那我们再来看下状态。

现在显示已经修改成功了！
Idea测试一下！

出现这个提示点击下载就可以了，idea会下载你连接mysql需要的jar包。

测试成功！
